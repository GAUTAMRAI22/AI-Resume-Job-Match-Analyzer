AI-Powered Resume & Job Match Analyzer 

An end-to-end AI-powered web application that analyzes resumes against job descriptions using NLP and embeddings to calculate match scores, identify skill gaps, and provide AI-driven improvement suggestions.
 Problem Statement

Job seekers often struggle to understand:
- Why their resumes get rejected
- Whether their resume matches a specific job description
- Which skills they need to improve to increase shortlisting chances

 Solution Overview

This application:
- Parses resumes (PDF/DOCX)
- Analyzes job descriptions
- Uses NLP and embeddings to compute resume–job match score
- Identifies missing skills and keyword gaps
- Suggests resume improvements optimized for ATS systems


Core Features

 MVP Features
- Resume upload (PDF/DOCX)
- Job description input
- Resume–job match score (%)
- Skill gap analysis

 Advanced Features
- AI-based resume rewrite suggestions
- ATS keyword optimization
- Job recommendations based on resume profile
- Resume history & improvement tracking
